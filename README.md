# TaskFlow - 暗黑风格任务管理应用

一款现代化的任务管理 Web 应用，采用暗黑主题设计，支持多页面、动态效果和本地数据存储。

---

## 📸 应用预览

| 仪表盘 | 任务列表 | 新建任务 |
|--------|----------|----------|
| 统计卡片、活跃任务、即将到期 | 状态筛选、分类过滤、搜索 | 表单填写、标签添加 |

## ✨ 功能特性

### 📊 仪表盘

- **统计卡片**：实时显示任务总数、已完成、进行中、逾期数量
- **活跃任务**：展示当前进行中的任务（最多 5 个）
- **即将到期**：未来 7 天内截止的任务提醒
- **快速搜索**：顶部搜索框支持关键词筛选

### ✅ 任务管理

| 功能 | 描述 |
|------|------|
| **创建任务** | 填写标题、描述、分类、优先级、截止日期、标签 |
| **编辑任务** | 支持修改所有任务属性 |
| **删除任务** | 二次确认弹窗，防止误删 |
| **完成/恢复** | 点击复选框切换状态，带有动画反馈 |

### 🏷️ 分类系统

- 预设 4 个默认分类：工作、个人、学习、其他
- 支持自定义添加、编辑、删除分类
- 每个分类可设置独特颜色
- 删除分类时，相关任务自动移至"未分类"

### 🔥 优先级管理

- **高优先级**：红色标识，紧急任务
- **中优先级**：黄色标识，普通任务（默认）
- **低优先级**：蓝色标识，非紧急任务
- 任务左侧彩色边条直观显示优先级

### 📅 截止日期

- 支持日期 + 时间精确设定
- 逾期任务自动标记红色警告
- 相对时间显示（今天、明天、3天后等）

### 🔍 筛选与搜索

- **状态筛选**：全部 / 进行中 / 已完成 / 已逾期
- **分类筛选**：下拉选择特定分类
- **优先级筛选**：按高/中/低优先级过滤
- **关键词搜索**：标题、描述、标签全文检索

### 💾 数据存储

- 使用浏览器 `localStorage` 本地存储
- 无需服务器，数据永不丢失
- 支持数据导入/导出（代码预留）

## 🎨 设计风格

### 暗黑主题

```
背景色层级：
├── 主背景  #0f0f0f  （最深）
├── 二级    #1a1a1a  （侧边栏、卡片）
├── 三级    #252525  （任务项、输入框）
└── 悬停    #2f2f2f  （交互反馈）

强调色：
├── 主色    #6366f1 → #818cf8 （渐变紫色）
├── 成功    #10b981 （绿色）
├── 警告    #f59e0b （橙色）
├── 错误    #ef4444 （红色）
└── 信息    #3b82f6 （蓝色）
```

### 动画效果

| 动画类型 | 应用场景 |
|----------|----------|
| `fadeIn` | 页面加载、元素出现 |
| `slideUp` | 卡片、列表滑入 |
| `slideInLeft` | 侧边栏、通知弹出 |
| `pulse` | 统计数字变化 |
| `taskAdd` | 新任务添加动画 |
| `taskDelete` | 任务删除消失动画 |
| `taskComplete` | 完成任务庆祝动画 |
| `shimmer` | 加载骨架屏 |
| `float` | Logo 浮动效果 |
| `gradientShift` | 渐变背景流动 |

### 响应式布局

- **大屏（>768px）**：侧边栏固定，内容区右侧展开
- **小屏（<768px）**：侧边栏隐藏，点击菜单图标展开
- **自适应卡片**：统计卡片、分类网格自动调整列数

## 🚀 快速开始

### 方式一：直接打开

用浏览器打开项目首页：

```
E:\todo-app\index.html
```

### 方式二：VS Code 运行

1. 用 VS Code 打开 `todo-app` 文件夹
2. 右键 `index.html` → 选择「在浏览器中打开」
3. 或安装 Live Server 插件，实现实时预览

### 方式三：本地服务器

```bash
# 使用 Python
cd todo-app
python -m http.server 8080

# 使用 Node.js
npx serve .
```

然后访问 `http://localhost:8080`

## 💡 使用指南

### 首次使用

系统自动初始化 4 个默认分类和空任务列表，可直接开始使用。

### 新建任务

1. 点击侧边栏「新建任务」或任务列表页顶部的 `➕ 新建任务` 按钮
2. 填写任务标题（必填）
3. 添加描述、选择分类、设置优先级、截止日期
4. 输入标签（按 Enter 添加）
5. 点击「创建任务」保存

### 完成任务

- 点击任务左侧的**圆形复选框**即可切换完成状态
- 完成后任务会显示绿色勾选、标题置灰、划线效果
- 可随时点击恢复为进行中状态

### 编辑/删除

- **悬停任务**：右上角出现 `✏️` 编辑和 `🗑️` 删除按钮
- **编辑**：跳转到编辑页面修改任务属性
- **删除**：弹出确认对话框，防止误删

### 分类管理

1. 点击侧边栏「分类管理」
2. 输入分类名称，选择颜色
3. 点击「添加分类」创建
4. 可编辑或删除已有分类
